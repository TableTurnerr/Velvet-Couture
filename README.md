# Velvet Couture Medspa

Version 1.0.0

Next.js static site rebuilt from the copied multi-route source structure and rebranded for Velvet Couture Medspa.

## Routes

- `/` homepage
- `/menu/` services
- `/specialties/` treatment spotlights
- `/specialties/[topic]/` treatment detail pages
- `/our-story/` about
- `/iraqi-cuisine/` aesthetic and wellness guide
- `/bakery/` skin care and facials
- `/catering/` events and training
- `/near/` Houston service areas
- `/return-policy/` appointment and booking policy

## Commands

```bash
npm run dev
npm run build
npx playwright install chromium
```
